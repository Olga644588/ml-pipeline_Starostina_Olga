Model Deployment Project

Настроен CI/CD-пайплайн для ML-сервиса с использованием Gitverse.
Обоснована и реализована стратегия развертывания Canary.
Спланировано A/B-тестирование для ML-модели.
Настроен CI/CD-деплоя через GitHub Actions: создан файл .github/workflows/deploy.yml, проведена проверка результата, приложены скриншоты.
